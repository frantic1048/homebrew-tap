# Homebrew Tap

Homebrew formulae for [frantic1048](https://github.com/frantic1048)'s projects.

## Available Formulae

| Formula | Description |
| ------- | ----------- |
| `blend` | Dotfiles manager powered by Nickel DSL ([repo](https://github.com/frantic1048/Vanilla/tree/master/blend)) |

## Install

```bash
brew install frantic1048/tap/blend
```

## Upgrade

```bash
brew upgrade blend
```

## Uninstall

```bash
brew uninstall blend
brew untap frantic1048/tap  # optional: remove the tap
```
